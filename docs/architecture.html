Factory Inventory Management System

System Architecture & Technical Reference

Tech Stack

Frontend

  • Vue 3 + Composition API
  • Vue Router 4
  • Vite 5 (port 3000)
  • Axios HTTP client
  • Custom i18n (EN / JP)

Backend

  • Python FastAPI
  • Uvicorn ASGI (port 8001)
  • Pydantic v2 models
  • CORS middleware

Data Layer

  • JSON flat files (server/data/)
  • In-memory loading at startup
  • No database (mock data)
  • 7 data files

Testing

  • pytest + FastAPI TestClient
  • httpx for async tests
  • pytest-cov coverage
  • 120+ backend tests

Key Features

Dashboard

KPI cards, inventory value, order health, low-stock alerts, shortage warnings with custom SVG charts.

Inventory Management

Searchable stock table with detail modals showing location, cost breakdown, and reorder points.

Order Tracking

Order list with status badges, inline item details, and delivery date tracking.

Demand Forecasting

Current vs. forecasted demand with trend indicators (increasing / stable / decreasing).

Spending Analytics

Revenue, costs, profit margins with monthly breakdowns and category analysis.

Internationalization

Full EN / JP language support with automatic currency switching (USD / JPY).
